# Samaan Sathi AI

AI-Powered Retail Management System for Small Shop Owners in India

## Quick Start

```powershell
cd samaan-sathi
.\deploy.ps1
```

Interactive menu with all options:
1. Deploy Backend
2. Fix Login Issue  
3. Confirm User
4. Test Locally
5. Deploy to S3
6. Get API URL

## Fix "User Not Confirmed" Error

```powershell
cd samaan-sathi
.\deploy.ps1
# Choose option 3
# Enter your username
```
